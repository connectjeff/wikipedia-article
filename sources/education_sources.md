# Education Source Review for Jeffrey Alderson

Scope: sources reviewed for education credentials and graduate coursework context.

## Bottom Line

The MathWorks speaker bio currently provides the clearest published source for the University of Illinois Urbana-Champaign Master of Education item. It is affiliated, so it should support only the narrow education fact and should not carry the AfC notability argument.

The University of Illinois Urbana-Champaign Digication portfolio is useful supporting context because it is hosted on the Illinois Digication system and identifies a published `Jeff Alderson Instructional Design` portfolio. The welcome page is the preferred entry point. A related page is titled `EPOL 483 Learning Technologies`. The fetched page metadata does not itself state that the degree was completed, so the portfolio should not replace the MathWorks speaker bio as the main degree citation.

Two private transcript PDFs are present in the project root: `UIUC Transcript 06MAY2022.pdf` and `Jeffrey Alderson WPI Transcript.pdf`. They are private academic records and should not be cited in the Wikipedia draft or included as public AfC evidence. The WPI transcript preview corroborates the WPI degree details. The UIUC file is a secured Parchment PDF from the University of Illinois at Urbana-Champaign; the available preview confirms it is an official transcript for Jeffrey Alan Alderson, but page-level degree details were not extracted in this pass. The public MathWorks speaker bio remains the draft citation for the M.Ed. item.
