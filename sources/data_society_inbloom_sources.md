# Data & Society / inBloom Source Review for Jeffrey Alderson

Scope: Data & Society and inBloom-related sources reviewed for the Jeffrey Alderson AfC proposal.

## Bottom Line

The Data & Society working paper is the strongest currently reviewed source for the article's education-technology notability argument. It is independent, research-oriented, and includes substantive discussion of inBloom with attributed commentary from Alderson. It is still not a biographical profile, so it should be framed as evidence of Alderson's role in a larger nationally discussed education-technology initiative rather than as profile coverage.
